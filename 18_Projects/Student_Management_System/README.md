# Student Management System (In-Memory)

A console application to add, view, and search student records, using an
array of structures to store data in memory during program execution.

## Concepts Used
- Structures
- Arrays of structures
- Functions
- String handling

## How to Run
```bash
gcc student_management.c -o student_management
./student_management
```

## Note
This version stores data in memory only (it resets each time the program
runs). See the [Library_Management_System](../Library_Management_System)
project for an example that persists data to a file.
