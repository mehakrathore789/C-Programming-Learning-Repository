# Number Guessing Game

The computer randomly picks a number between 1 and 100. The player has to
guess it, receiving "too high"/"too low" hints after each attempt.

## Concepts Used
- `rand()` and `srand()` for random number generation
- Loops (`while`)
- Conditional statements

## How to Run
```bash
gcc guessing_game.c -o guessing_game
./guessing_game
```
