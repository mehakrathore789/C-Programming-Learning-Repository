# 18. Projects

Mini projects that combine everything learned in previous sections —
functions, arrays, structures, strings, loops, and file handling — into
complete, runnable programs. These are great portfolio pieces.

## Projects

| Project | Concepts Used | Description |
|---------|----------------|--------------|
| [Simple_Calculator](./Simple_Calculator) | switch, functions | A menu-driven calculator for basic arithmetic |
| [Number_Guessing_Game](./Number_Guessing_Game) | loops, rand(), conditionals | Guess a randomly generated number with hints |
| [Student_Management_System](./Student_Management_System) | structs, arrays, functions | Add, view, and search student records (in-memory) |
| [Library_Management_System](./Library_Management_System) | structs, file handling | Persist book records to a file on disk |

Each project folder contains its own `README.md` with instructions on how
to compile and run it, along with the source code.
