# 03. Operators

Operators are symbols that perform operations on variables and values.

## Concepts Covered
- Arithmetic operators (`+ - * / %`)
- Relational operators (`== != > < >= <=`)
- Logical operators (`&& || !`)
- Bitwise operators (`& | ^ ~ << >>`)
- Assignment operators (`= += -= *= /= %=`)
- Ternary (conditional) operator (`?:`)
- Increment / Decrement operators (`++ --`)

## Key Takeaways
- `%` (modulus) only works with integers, not floats.
- `&&` and `||` short-circuit: the second operand may not be evaluated.
- Bitwise operators work at the binary level and are common in embedded/systems programming.
- Pre-increment (`++x`) increments before use; post-increment (`x++`) increments after use.
