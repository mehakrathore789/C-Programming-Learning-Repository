# 07. Pattern Printing Programs

Pattern printing is a classic exercise for mastering nested loops.
Each program below prints a different shape using stars (`*`) or numbers.

## Files

| File | Description |
|------|--------------|
| `01_square_pattern.c` | A solid square of stars |
| `02_right_triangle_pattern.c` | A right-angled triangle of stars |
| `03_pyramid_pattern.c` | A centered pyramid of stars |
| `04_inverted_pyramid_pattern.c` | An upside-down pyramid |
| `05_diamond_pattern.c` | A diamond made of two pyramids |
| `06_number_pattern.c` | A triangle built from increasing numbers |
| `07_floyds_triangle.c` | Floyd's Triangle — consecutive numbers arranged in a triangle |

## Key Takeaways
- The outer loop controls the number of rows.
- The inner loop(s) control what is printed in each row (spaces and/or symbols).
- Pattern problems are one of the best ways to build intuition for nested loops.
