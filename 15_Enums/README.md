# 15. Enumerations (enum)

An `enum` defines a set of named integer constants, making code more
readable than using plain "magic numbers".

## Files

| File | Description |
|------|--------------|
| `01_enum_basics.c` | Declaring and using an enum |
| `02_enum_in_switch.c` | Using enums with switch-case for readable state handling |

## Key Takeaways
- By default, enum values start at 0 and increase by 1 for each member.
- You can assign specific values to enum members.
- Enums make code self-documenting compared to raw integers.
