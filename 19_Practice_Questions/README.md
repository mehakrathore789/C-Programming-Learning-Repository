# 19. Practice Questions

A set of solved practice problems that combine concepts from across the
repository, similar to what you'd encounter in coding interviews or coursework.
Try to solve each one yourself before checking the solution!

## Files

| File | Description |
|------|--------------|
| `01_swap_two_numbers.c` | Swap two numbers with and without a temporary variable |
| `02_prime_number_check.c` | Check whether a number is prime |
| `03_armstrong_number.c` | Check whether a number is an Armstrong number |
| `04_bubble_sort.c` | Sort an array using the Bubble Sort algorithm |
| `05_binary_search.c` | Search a sorted array using Binary Search |
| `06_count_words_in_string.c` | Count the number of words in a sentence |
| `07_matrix_transpose.c` | Compute the transpose of a matrix |
| `08_check_leap_year.c` | Determine whether a given year is a leap year |

## Key Takeaways
- These problems are frequently asked in technical interviews — practice
  explaining your approach out loud, not just writing the code.
- Try to identify the time complexity of each solution.
