# 01. Basics of C

This section introduces the fundamental building blocks of a C program.

## Concepts Covered
- Structure of a C program (headers, `main()`, return statement)
- Comments (single-line and multi-line)
- Variables and how to declare them
- Constants (`const` keyword and `#define`)
- Keywords and identifiers

## Files

| File | Description |
|------|--------------|
| `01_hello_world.c` | The classic first program — prints "Hello, World!" |
| `02_program_structure.c` | Explains headers, main function, and return statement |
| `03_comments.c` | Single-line and multi-line comments |
| `04_variables.c` | Declaring and initializing variables |
| `05_constants.c` | `const` keyword and `#define` macro constants |
| `06_keywords_identifiers.c` | Reserved keywords vs valid identifiers |

## Key Takeaways
- Every C program must have exactly one `main()` function — execution starts here.
- `#include` statements bring in library functionality (like `stdio.h` for input/output).
- Statements end with a semicolon `;`.
- C is case-sensitive: `Total` and `total` are different identifiers.
