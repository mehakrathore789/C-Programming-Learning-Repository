# 06. Loops

Loops let you repeat a block of code multiple times without duplicating it.

## Concepts Covered
- `for` loop — best when the number of iterations is known
- `while` loop — repeats while a condition remains true
- `do-while` loop — always runs at least once
- Nested loops — a loop inside another loop
- `break` and `continue` — controlling loop flow

## Key Takeaways
- `for` loop syntax: `for (initialization; condition; update)`.
- A `do-while` loop always executes its body **at least once**, unlike `while`.
- `break` exits the loop entirely; `continue` skips to the next iteration.
