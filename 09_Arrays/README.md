# 09. Arrays

An array is a collection of elements of the same type, stored in contiguous memory.

## Concepts Covered
- Declaring and initializing 1D arrays
- Traversing and modifying array elements
- Common array operations (sum, max/min, search, reverse)
- 2D arrays (matrices)
- Multidimensional array basics

## Key Takeaways
- Array indices start at **0**, not 1.
- The array name itself acts as a pointer to its first element.
- C does not check array bounds — accessing out-of-range indices is undefined behavior.
